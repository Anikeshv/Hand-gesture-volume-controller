# Hand Gesture Volume Controller 🎵✋

A Python + OpenCV project that allows users to control system volume in real time using hand gestures.  
By detecting and tracking hand landmarks, the system provides an intuitive, hands-free way to adjust audio levels.

---

## 🚀 Features
- Real-time hand tracking using OpenCV.
- Gesture recognition for volume up, down, and mute.
- Smooth integration with system audio controls.
- Lightweight and easy to run on most devices.

---

## 🛠 Tech Stack
- **Python**
- **OpenCV** (Computer Vision)
- **Pycaw** (Audio control library)
